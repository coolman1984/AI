# Extracted Requirements (Mission Analyst)
Real goal: an operational decision brain for a factory that reads files, computes exact
numbers, remembers relations + change over time, and presents audited, human-approved
management answers. Full requirements live in `MASTER_PLAN.md` (Parts A–S) and
`FACTORY_SECOND_BRAIN.md`. This file indexes them; it does not duplicate them.
- Explicit: extract/clean/store/analyze/search/report/dashboard; one dept first, then factory.
- Implied: trust (no confident wrong number), in-house/offline, reviewability, restartability.
- Non-negotiable: independent audit + named human sign-off before anything reaches the CEO.
