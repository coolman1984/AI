# Terminology
- Lens: a department config (lenses/*.yaml) — adds a department without engine code.
- Variance bridge: actual - baseline by grain; must reconcile to the total.
- Evidence ref: source+locator+method behind every number (trust wall).
- Audit: independent re-run (four-eyes) + certainty + human questions + sign-off.
- needs_review: low-confidence OCR/quality -> human-review queue (never silently trusted).
